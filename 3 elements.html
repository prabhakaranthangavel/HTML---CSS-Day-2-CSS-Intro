Inline Elements & Block Elements

Block Elements:

Have only one tag used is called block elements.

Eg:

romanreignsjhoncenatheundertaker

h1 tag

div tag

paragraph tag

anchor tag


Inline Elements:

Inside the block scope another one or more tag used is called inline elements.

Eg:

bold tag

strong tag

span tag

romanreignstherockjhoncenatheundertaker
